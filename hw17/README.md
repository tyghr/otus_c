Игра

Цель:
- Получить опыт создания игр.

Сложность
★★★★☆

Описание домашнего задания:
- Написать несложную игру, на выбор:
    - тетрис
    - сапёр
    - 2048
    - крестики-нолики
    - четыре-в-ряд
    - ping-pong
    - space invaders
    - pacman

Требования:
- Создано игровое приложение с главным меню и основным gameplay loop.
- Бонусные баллы за графическое оформление с использованием ассетов.
- Бонусные баллы за звуковое оформление.
- Бонусные баллы за таблицу лидеров.
- Код компилируется без предупреждений с ключами компилятора -Wall -Wextra -Wpedantic -std=c11.

Вспомогательные материалы:
- [Как начать писать игры](https://habr.com/ru/post/160547/)

---------------------------------

## Решение:

```sh
sudo apt-get install libsdl2-dev

make
./app

```
